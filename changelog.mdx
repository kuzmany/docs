<Update label="2025-02-28" tags={["Multiple Transport"]}>

  ```bash
  composer require kuzmany/mautic-multiple-transport-bundle:^2.0
  ```

  This command will add the Mautic Multiple Transport Bundle (version 2.0 or higher) to your project. More details at [Multiple Transport](https://mtcextendee.com/mautic-multiple-transport-bundle).

<Update label="2025-01-25" tags={["Twig Templates"]}>
  Enable support for embedding Twig templates within other Twig templates for Mautic 5.
  ```twig
  {% if contact.gender == 'm' %}  
  {twigtemplate=helloM}  
  {% else if contact.gender == 'w' %}  
  {twigtemplate=helloW}  
  {% endif%}
  ```
  More details at [Twig Templates](https://mtcextendee.com/mautic-twig-templates-bundle).
</Update>
