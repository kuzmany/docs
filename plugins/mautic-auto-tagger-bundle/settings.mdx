# Auto Tagger Plugin Settings

This page describes the configuration options available for the Mautic Auto Tagger plugin.

---

## Accessing the Plugin Settings

1. Go to **Settings** (cog icon) > **Plugins** in your Mautic instance.
2. Find **Automatic UTM Tagger** and click on it.
3. Click the **Features** tab.

---

## Options

- **Automatically add UTM tags during creating new form**  
  If enabled, default UTM tags are pre-filled when creating a new Email, Dynamic Content, or Focus Item.

- **Automatically append UTM parameters to URLs if not exist**  
  If enabled, the plugin will automatically add the configured UTM tags (and custom tags) to links found in the content if they don't already have UTM parameters.

---

## Tags for Each Channel

You can configure default UTM tags and custom tags for:

- **Emails**
- **Dynamic Content**
- **Focus Items**

Each channel has its own section for UTM and custom tags.

---

## Using Tokens in Tags

You can use tokens in your tag values to insert dynamic data from Mautic entities.  
See the [introduction](./introduction) for token formats and examples.

---

## Saving Settings

After configuring your options and tags, click **Save & Close** to apply your changes.
