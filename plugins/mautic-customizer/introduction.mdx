Compatible with **Mautic 5** and **Mautic 6**

## Introduction

A simple plugin for Mautic that allows administrators to customize the Mautic user interface by hiding specific menu items and campaign or form elements.

## Features

- Hide items in the main navigation menu.
- Hide items in the admin settings menu.
- Hide actions, conditions, and decisions in the campaign builder.
- Hide submit actions in the form builder.
