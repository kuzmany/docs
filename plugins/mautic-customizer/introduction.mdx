A simple plugin for Mautic 5/6 that allows administrators to customize the Mautic user interface by hiding specific menu items and campaign builder elements.

## Features

*   Hide items in the main navigation menu.
*   Hide items in the admin settings menu.
*   Hide actions, conditions, and decisions in the campaign builder.
