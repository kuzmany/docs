A simple plugin for Mautic 5/6 that allows administrators to customize the Mautic user interface by hiding specific menu items and campaign builder elements.

## Features

*   Hide items in the main navigation menu.
*   Hide items in the admin settings menu.
*   Hide actions, conditions, and decisions in the campaign builder.

## Configuration

1.  Install the plugin.
2.  Go to **Settings** (cog icon) > **Plugins**.
3.  Click the **Customizer** integration.
4.  Enable the integration by setting **Published** to "Yes".
5.  Go to the **Features** tab.
6.  Select the items you wish to hide in the "Hide Main Menu Items", "Hide Admin Menu Items", "Hide Campaign Actions", "Hide Campaign Conditions", and "Hide Campaign Decisions" fields.
7.  Save the configuration.

The selected menu items and campaign elements will now be hidden from users.
