Compatible with Mautic 4, 5, and 6.
</Note>

## Features

- Manage unlimited email transports.
- Add any Mautic-supported transport (SMTP, Mailjet, Amazon SES, Sendgrid, etc).
- Assign custom transport to each email.
- Use contact owner's transport for sending.
- Compatible with Mautic 4 (SMTP only), Mautic 5/6 (all transports).

## How to install

1. Copy the plugin to `plugins/YourBundleName`
2. Clear cache:  
   `php bin/console cache:clear`
3. Reload plugins list:  
   `php bin/console mautic:plugins:reload`
4. Regenerate assets:  
   `php bin/console mautic:assets:generate`

## FAQ

**How can I send emails with the contact owner's transport?**  
Create transports and attach them to the owner (user). Enable this option in email settings.

**Can I use a different transport other than SMTP?**  
Yes, any Mautic-supported transport is available.
