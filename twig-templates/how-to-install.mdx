# Installing Twig Templates

Follow these steps to install the Mautic Twig Templates bundle:

1.  **Download:** Once you receive the purchase confirmation email, use the provided link to download the latest version of the bundle.
2.  **Upload:** Copy the downloaded plugin folder to your Mautic installation's `plugins/` directory. Ensure the folder name is descriptive, like `MauticTwigTemplatesBundle`.
3.  **Clear Cache:** Run the following command from your Mautic root directory in the console:
    ```bash
    php bin/console cache:clear
    ```
4.  **Reload Plugins:** Run this command to make Mautic recognize the new plugin:
    ```bash
    php bin/console mautic:plugins:reload
    ```
5.  **Generate Assets:** Finally, regenerate assets with this command:
    ```bash
    php bin/console mautic:assets:generate
    ```

After these steps, the plugin should be installed and available within your Mautic instance.
